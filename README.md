## Korra

Korra is a social platform designed for pet lovers to connect, share, and explore pet-related content. Korra fosters a community where users can create profiles, post updates, and interact with content centered around their furry (or not-so-furry) friends.

## Features

* User Profiles: Showcase your pets and interests.
* Feed: Share updates, photos, and experiences with the community.
* Networking: Connect with other pet lovers and professionals.
* Marketplace: Buy or sell pet related items.

## Tech Stack

* Next.js
* Supabase
* Hono
* Prisma
* Additional Tools: React Query, Zod, React Hook Forms
